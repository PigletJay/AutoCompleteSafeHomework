## 安装

这个项目需要先配置好 `Java` 运行环境，请确保本地已配置它。

## 使用说明

> 运行后会在目录下生成 `students.txt` 文件

> 在文件内输入学生账号 **（每行一个）** 并保存

> 然后按下 **空格键** 等待自动完成即可
